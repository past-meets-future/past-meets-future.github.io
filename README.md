# Past Meets Future — workshop website

Static site for the Past Meets Future workshop series (Human-AI Interaction for Digital Humanities & Cultural Heritage).
Current edition: **/2026/** (HCOMP ⊕ CI 2026, Alexandria VA, Sun Sept 27 2026 · papers due July 30).

- Plain HTML/CSS + one small carousel script. No build step. Deploy: GitHub Pages, deploy-from-branch, `.nojekyll`.
- `2026/data/*.yml` are the canonical content files — edit data first, then mirror into `2026/index.html` (map in MAINTENANCE.md).
- Past years (`/2024/`, `/2025/`) are frozen archive pages. Never edit them.
- Images: CC0/public-domain only, self-hosted, never hotlinked, never filtered. Full credits in the page colophon.
- Brand & design rules: `DESIGN.md`. Plan & facts: `PLAN.md`. Working conventions: `CLAUDE.md`.
